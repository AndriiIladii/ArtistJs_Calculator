# Advanced Calculator

Learning project "Advanced Calculator"

## Features

- [ ] Basic Calculator
  - [ ] Enter and delete numbers
  - [ ] Basic arithmetic operations (+, -, \*, /)
  - [ ] Decimal numbers
  - [ ] Negative numbers
  - [ ] Percent button (%)
  - [ ] Square root
- [ ] Advanced features
  - [ ] Power ($x^{2}$ and $x^{n}$)
  - [ ] Trigonometry (sin, cos, tan)
  - [ ] Keyboard support
  - [ ] History
